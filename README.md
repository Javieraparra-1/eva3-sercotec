 Página Web SERCOTEC Santiago

¿Qué tiene la página?
- Un menú de navegación arriba con links a cada sección
- Una sección de bienvenida con un botón de contacto
- Información sobre quiénes somos y cuántas empresas hemos ayudado
- Tarjetas con los servicios que ofrecemos
- Noticias recientes del centro
- Próximos eventos y talleres
- Testimonios de empresarios que hemos apoyado
- Preguntas frecuentes
- Formulario de contacto
- Pie de página con dirección y redes sociales
¿Cómo se hizo?
Se usó Next.js con Tailwind CSS para el diseño. Cada sección de la página es un componente separado que se puede reutilizar.
¿Cómo instalarlo?
1. Descargar el proyecto
2. Abrir la terminal y escribir `npm install`
3. Luego escribir `npm run dev`
4. Abrir el navegador en `http://localhost:3000`
 Colores usados
- Azul: secciones principales
- Naranja: botones
- Gris oscuro: navbar y footer
- Beige/café claro: fondo general
